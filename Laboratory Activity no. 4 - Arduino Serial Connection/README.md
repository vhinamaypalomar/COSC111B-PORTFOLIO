## Objectives

## What Was Implemented
- Photoresistor-based brightness detection
- LED blinking behavior
- Serial command handling

## Code Summary
- Initializes Serial communication at 9600 baud
- Sends sensor readings to the Serial Monitor
- Reads text-based Serial commands
- Allows user input to override automated behavior
